## 1. Identidade Visual

A identidade visual é de alto contraste, utilizando fundos escuros para navegação e fundos claros para conteúdo, com o dourado como cor de destaque principal.

#### Paleta de Cores

A paleta é dividida por contexto:

| Uso / Contexto | Cor | Código Hex |
| :--- | :--- | :--- |
| **Fundo (Header e Footer)** | Preto | `#000000` |
| **Fundo (Páginas / Conteúdo)**| Branco | `#FFFFFF` |
| **Texto (Geral)** | Preto | `#000000` |
| **Header (Título e Setas)** | Dourado (Header) | `#FFCC3C` |
| **Footer (Ícones Ativos)** | Dourado (Footer) | `#F0B100` |
| **Botões (Confirmação)** | Dourado (Botão) | `#FCE444` |
| **Botões (Login/Auth Fundo)** | Cinza Escuro | `#212121` |
| **Botões (Login/Auth Texto)** | Dourado (Login) | `#F0B100` |

## 2. Tipografia

* **Fonte Principal:** **Poppins**
* **Variações de Peso:** A aplicação utiliza múltiplas variações da família Poppins, incluindo Light, Regular, SemiBold e Bold.

## 3. Layout e Componentes

O layout padrão das telas internas (pós-login) é composto por três seções:

1.  **Header (Cabeçalho)**
    * **Fundo:** Preto (`#000000`).
    * **Conteúdo:** Título da página e setas de navegação.
    * **Cor do Título:** Dourado (`#FFCC3C`).
    * **Tipografia:** Poppins SemiBold, 16pt.

2.  **Área de Conteúdo (Página)**
    * **Fundo:** Branco (`#FFFFFF`).
    * **Conteúdo:** Ocupa o espaço entre o Header e o Footer.
    * **Cor do Texto:** Preto (`#000000`).

3.  **Footer (Navegação por Abas)**
    * **Fundo:** Preto (`#000000`).
    * **Conteúdo:** Ícones de navegação principal.
    * **Cor dos Ícones:** Dourado (`#F0B100`).
    * **Tipografia (Rótulos):** Poppins Regular, 10pt.
